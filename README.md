# FeatherClient离线缓存端
## 解决某些人无法下载FeatherClient
## 适用情况
* 启动游戏卡在下载libraries的
* 按下启动按钮一会儿又恢复原样的
* 无梯子又想下载Feather Client的游戏文件的
* 游戏启动报错需要修复替换的

* 下载压缩包并解压出来（至一个文件夹内）
* 在Feather启动器里选择解压出来的文件夹
* 启动享受吧！
